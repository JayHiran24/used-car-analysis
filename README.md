\# Used Car Market Analysis



Exploratory data analysis of the Indian used car market using the Car Dekho dataset.



\*\*Tools:\*\* Python, Pandas, Matplotlib, Seaborn  

\*\*Dataset:\*\* \[Car Details from Car Dekho](https://www.kaggle.com/datasets/nehalbirla/vehicle-dataset-from-cardekho)



\## Key Questions

\- Which factors most influence resale price?

\- How does km driven affect value by fuel type?

\- Which brands retain value best over time?
